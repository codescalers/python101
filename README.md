## Setup

Install the following:

+ [Virtualbox](https://www.virtualbox.org/wiki/Downloads) 
+ [Vagrant](https://www.vagrantup.com/downloads.html)

then run the following command from this repo

```
$ vagrant up
```

to get access to the machine

```
$ vagrant ssh
```
